# imageio API Reference

Documentation for the imageio library API, exposing all public classes and their methods and fields

### Package `imageio.image`
This module includes the `Image` class which holds pixel data, as well as the pixel classes

* [Image](api/Image.md)
* [RGBA8](api/RGBA8.md)
* RGBA16
* RGBA32

### Package `imageio.wrapper`
This module handles the ImageMagick wrapping. From `magick` class you can read an image file

* [magick](api/magick.md)
* process
